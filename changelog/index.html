v2.3.0

2026-03-28
ToolbarUXExport
  • Introduced a two-layer top toolbar model: primary tools on line 1 and support tools on line 2.
  • Added support toolbar visibility modes: Auto, Show, and Hide (persisted per browser).
  • Support toolbar now appears as an overlay dropdown in Auto mode so canvas/work area does not shift on hover.
  • Promoted diagram export actions (SVG, PNG, Copy PNG) to the primary toolbar for faster access.
